## Installation guide

Step 1: Install using npm all modules related

```
npm install
```

Step 2: Check .env.example file for configurating your DB

```
KEY = VALUE

DB_HOST=
DB_USER=
DB_PASSWORD=
DB_NAME=
DB_PORT=
```

Step 3: Use.
